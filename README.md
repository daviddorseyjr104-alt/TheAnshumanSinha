# Anshuman Sinha

Personal site for Anshuman Sinha, entrepreneur, angel investor, and startup mentor. "I help founders pitch, fund and scale."

A single page static site built with plain HTML, CSS, and vanilla JavaScript, with Lenis and GSAP for smooth scrolling and motion. Sleek dark editorial design, real photos and video, and interactive sections (process, testimonials carousel, tabs, FAQ accordion, timeline, marquees). Fully responsive, keyboard accessible, and reduced motion friendly, with graceful fallbacks if the animation libraries or JavaScript are unavailable.

## Run locally

Serve the folder over HTTP, for example:

    python -m http.server 4936

Then open http://localhost:4936

## Deploy

This is a static site, so any static host works (GitHub Pages, Netlify, Vercel, Cloudflare Pages). For GitHub Pages, enable Pages on the `main` branch from the root folder, and the site will be served from `index.html`.

## Structure

    index.html    markup and content
    styles.css    design system and layout
    script.js     motion and interactivity
    images/       real photos, hero video, logo
